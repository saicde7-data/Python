# Navigation Shortcuts

## Navigate to File/Class/Symbol:

- Open the "Navigate" menu using `Ctrl + N` (Windows/Linux) or `Cmd + O` (Mac). You can then start typing the name of
  the file, class, or symbol you want to navigate to.

## Navigate to Declaration/Definition:

- Place the cursor on a symbol and use `Ctrl + B` (Windows/Linux) or `Cmd + B` (Mac) to jump to its declaration or
  definition.

# Editing Shortcuts

## Code Completion:

- Trigger code completion with `Ctrl + Space`.

## Quick Documentation:

- Display documentation for a symbol with `Ctrl + Q` (Windows/Linux) or `F1` (Mac).

## Reformat Code:

- Use `Ctrl + Alt + L` (Windows/Linux) or `Cmd + Alt + L` (Mac) to reformat code. PyCharm offers various refactorings
  accessible through the right-click context menu.

## Surround with Code Block:

- Select code and use `Ctrl + Alt + T` (Windows/Linux) or `Cmd + Alt + T` (Mac) to surround it with a code block (if,
  try, while, etc.).

# Version Control Integration

## VCS Operations:

- Access version control operations with `Alt + 9` to open the Version Control tool window.

## Commit Changes:

- Commit changes using `Ctrl + K` (Windows/Linux) or `Cmd + K` (Mac).


# Productivity Features

## Quick Search Everywhere:

- Use `Shift + Shift` to search everywhere - files, classes, symbols, actions, and more.


## Terminal:

- Open the built-in terminal with `Alt + F12`.

These are just a few of the many shortcuts and features available in PyCharm. Explore the IDE, and customize your keymap
according to your preferences through `File -> Settings -> Keymap` (or `PyCharm -> Preferences -> Keymap` on macOS).
